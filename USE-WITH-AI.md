# AI에게 복붙하기 (교사용)

만든 앱 폴더를 열어둔 상태에서, 쓰고 있는 AI(Claude Code·Cursor 등)에게 아래를 그대로 붙여넣으세요. 개발을 몰라도 됩니다. AI가 검사하고, 무엇을 왜 고쳐야 하는지 쉬운 말로 알려주고, 당신 동의 하에 고쳐줍니다.

---

내 앱의 보안과 개인정보를 dorms-check 로 점검하고, 도름스 인증마크를 받을 수 있는 상태까지 함께 고쳐줘.

1. 먼저 이렇게 알려줘: 이 도구는 코드를 대신 고치거나 인증을 발급하지 않아. 무엇이 안전하고 무엇을 고쳐야 하는지 알려주는 코치야. 최종 마크는 도름스 서버가 스스로 다시 검사해서 발급해.
2. `npx -y dorms-check@latest detect` 로 스택을 확인해줘.
3. `npx -y dorms-check@latest init --name "내 앱 이름" --url "내 배포 주소" --track security,edzip --confirm-ownership` 로 설정을 만들어줘. (학운위 준비까지 원하면 track 에 edzip 을 넣고, 학생 개인정보를 다루는지 3문항을 나한테 물어봐줘.)
4. `npx -y dorms-check@latest scan --url "내 배포 주소"` 로 검사해줘.
5. 검사가 "AI가 판단해야 할 항목"을 알려주면, 네가 내 코드와 개인정보처리방침을 직접 읽고 판단해서 `judge` 로 증거(파일:라인)와 함께 기록해줘. 증거 없이 통과 처리하지 마.
6. `npx -y dorms-check@latest status` 로 남은 항목을 보고, 각 항목을 왜 고쳐야 하는지 나한테 쉬운 말로 설명한 다음, 내 동의를 받고 내 앱 코드를 고쳐줘.
7. 고친 뒤 다시 배포하고 4~6을 반복해줘. 보안과 학운위 준비가 모두 통과할 때까지.
8. 다 통과하면 `npx -y dorms-check@latest submit` 으로 증빙팩을 만들고, 도름스에서 마크 신청하는 방법을 알려줘.

중요: 읽기 검사만 하고(비파괴), 내 앱만 검사해. "대충 통과"시키지 말고, 실제로 확인된 것만 통과로 처리해줘.

---

이게 전부입니다. 붙여넣고 AI가 안내하는 대로 따라가면 됩니다.
